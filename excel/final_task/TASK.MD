# Задание: Аналитика продаж с использованием Pandas и OpenPyXL


## Требования

### 1. Входные данные
- **Файл**: `sales_data.csv`
- **Структура данных**:
  - `order_id` - уникальный идентификатор заказа
  - `country` - страна
  - `category` - категория товара (Electronics, Furniture)
  - `price` - цена товара
  - `quantity` - количество товара
  - `date` - дата заказа

### 2. Обработка данных

#### 2.1 Основные вычисления
- Расчёт выручки по каждому заказу: `revenue = price × quantity`
- Общие метрики:
  - Количество заказов
  - Общая выручка
  - Средняя стоимость заказа
  - Среднее количество товара в заказе

#### 2.2 Агрегация данных

**По странам** (group by country):
- Общая выручка по стране
- Средняя стоимость заказа
- Количество заказов
- Средняя цена товара
- Среднее количество товара

**По категориям** (group by category):
- Общая выручка по категории
- Средняя стоимость заказа
- Количество заказов
- Общее количество товара

#### 2.3 Статистические показатели
Для столбца `price` рассчитать:
- Минимальное значение
- Максимальное значение
- Среднее значение
- Медиану
- Стандартное отклонение

### 3. Вывод результатов

#### 3.1 Консоль
- Информативный вывод с структурированием по этапам
- Таблицы агрегированных данных
- Статистические показатели

#### 3.2 Excel-файл (`sales_report.xlsx`)

Файл должен содержать единый лист "Аналитика Продаж" с разделами:

**Раздел 1: Заголовок и дата**
- Название отчёта: "ОТЧЁТ ПО ПРОДАЖАМ"
- Дата и время создания

**Раздел 2: Общие метрики**
- Всего заказов
- Общая выручка
- Средняя стоимость заказа
- Среднее количество товара в заказе

**Раздел 3: Анализ по странам**
- Таблица с колонками:
  - Страна
  - Общая выручка
  - Средний заказ
  - Количество заказов
  - Средняя цена

**Раздел 4: Анализ по категориям**
- Таблица с колонками:
  - Категория
  - Общая выручка
  - Средний заказ
  - Кол-во заказов

#### 3.3 Форматирование

**Заголовки (Headers)**:
- Шрифт: Calibri, размер 12, жирный, белый текст
- Фон: синий (#4F81BD)
- Выравнивание: центр
- Границы: тонкие линии

**Заголовок отчёта**:
- Шрифт: Calibri, размер 14, жирный, белый текст
- Фон: тёмно-синий (#203864)
- Объединённые ячейки по ширине

**Заголовки разделов**:
- Шрифт: Calibri, размер 11, жирный, белый текст
- Фон: светло-голубой (#8FAADC)
- Объединённые ячейки

**Данные**:
- Финансовые значения: формат валюты с символом ₽
- Выравнивание: вправо
- Границы: тонкие линии

**Ширина столбцов**:
- Колонка A (названия): 20
- Остальные колонки: 18

## Навыки, демонстрируемые проектом

### Python
- Работа с функциями и модулями
- Работа с циклами и условиями
- Форматирование строк (f-strings)

### Pandas
- Чтение CSV файлов (`read_csv`)
- Создание новых колонок
- Агрегация данных (`groupby`, `agg`)
- Статистические расчёты (`sum`, `mean`, `min`, `max`, `median`, `std`)
- Сортировка данных (`sort_values`)
- Работа с DataFrames

### OpenPyXL
- Создание Excel workbooks
- Форматирование ячеек (шрифты, цвета, выравнивание)
- Применение границ и стилей
- Работа с объединённой ячейками
- Установка ширины колонок
- Форматирование чисел (валюта)
- Сохранение файлов
